# Simulador-de-processador-em-C

Projeto desenvolvido para a disciplina de **Arquitetura de Computadores**.

O objetivo deste trabalho é simular o funcionamento interno de uma CPU, implementando em software os conceitos de hardware estudados em aula. O simulador foi escrito em **Linguagem C** e reproduz o ciclo de instrução (Busca, Decodificação e Execução).

### Funcionalidades
* Simulação de memória RAM e registradores.
* Execução de instruções aritméticas e lógicas.
* Controle de fluxo (desvios condicionais).
* Manipulação correta de tipos de dados de 16 bits (uso de `uint16_t` para endereçamento).
